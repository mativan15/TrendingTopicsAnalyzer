The privacy-minded DuckDuckGo leads a full life as a search engine, a browser extension and a mobile app. But now it's also accessible as a dedicated web browser for the Mac. With the DuckDuckGo browser for macOS, you can block third-party trackers, disable cookies, turn on HTTPS for websites and tighten other security and privacy via the browser settings. Let's check it out. DuckDuckGo is now in beta testing for macOS with a Windows version promised later in 2022. As such, you currently must request an invitation to use the Mac version.
