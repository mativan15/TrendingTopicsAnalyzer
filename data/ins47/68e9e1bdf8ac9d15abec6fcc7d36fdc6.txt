Many investors are still learning about the various metrics that can be useful when analysing a stock. This article is for those who would like to learn about Return On Equity (ROE). We'll use ROE to examine Journeo plc (LON:JNEO), by way of a worked example.

Return on equity or ROE is a key measure used to assess how efficiently a company's management is utilizing the company's capital. In short, ROE shows the profit each dollar generates with respect to its shareholder investments.

How Do You Calculate Return On Equity?

The formula for return on equity is:

Return on Equity = Net Profit (from continuing operations) ÷ Shareholders' Equity
