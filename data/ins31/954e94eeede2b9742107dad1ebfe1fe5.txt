NEW YORK, Oct. 17, 2022 /PRNewswire/ -- Tuya Smart (NYSE: TUYA, HKEX: 2391)("Tuya"), a global IoT development platform service provider, announced today that it has joined the Amazon Web Services (AWS) Independent Software Vendor (ISV) Accelerate Program, a co-sell program for AWS Partners that provides software solutions that run on or integrate with AWS. The program helps AWS Partners drive new business by directly connecting participating ISVs with the AWS Sales organization.
The AWS ISV Accelerate Program provides Tuya with co-sell support and benefits to meet customer needs through collaboration with AWS field sellers globally.
