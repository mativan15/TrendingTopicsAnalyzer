 MacBooks (along with Windows and Linux PCs) can now get a new operating system, but it's probably not the one you were expecting – Windows 95 has just risen from the dead in the form of a new app.As the Betanews website reports (opens in new tab), Felix Rieseberg, a Slack developer, has made a working version of Windows 95 as an Electron app (opens in new tab) using JavaScript. And with the latest version you can run it on almost any Windows computer, along with macOS-running MacBooks (including M1-powered ones) and Linux devices as well.
